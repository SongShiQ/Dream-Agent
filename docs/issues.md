# 项目问题清单

**创建时间**: 2026-07-11 16:10

---

## 问题列表

### 1. 按钮无功能 (严重)
**位置**: `app/page.tsx`
**问题**: 快捷功能按钮（评估水平、开始练习、查看进度、学习计划）没有 onClick 事件处理器
**影响**: 用户点击无反应，核心功能不可用

### 2. 无用户系统 (严重)
**问题**: 没有登录/注册功能，无法识别用户身份
**影响**: 无法存储个人学习数据，无法实现个性化

### 3. 无状态管理 (严重)
**问题**: 没有全局状态管理（如 React Context）
**影响**: 组件间无法共享数据，用户体验断裂

### 4. API 路由不完整 (中等)
**问题**: 只有 `/api/chat` 和 `/api/exam`，缺少：
- `/api/assess` - 评估 API
- `/api/progress` - 进度查询 API
- `/api/plan` - 学习计划 API
**影响**: 多个功能无法正常工作

### 5. 数据库未实际使用 (中等)
**问题**: SQLite 已创建，但代码中仍使用模拟数据
**影响**: 数据不持久化，刷新后丢失

### 6. 无会话持久化 (中等)
**问题**: 对话记录只在内存中，刷新页面丢失
**影响**: 用户体验差，无法回顾历史

### 7. 知识库未接入 (低)
**问题**: 知识库检索返回模拟数据
**影响**: 答案质量低，无实际参考价值
