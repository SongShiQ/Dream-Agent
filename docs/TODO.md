## 功能增强

### 1. API 切换功能 (优先级：高)
**描述**：像 ChatGPT 那样，让用户自由切换不同的 LLM API

**需求**：
- 设置页面，配置 API Provider 和 Key
- 支持 OpenAI、Anthropic、DeepSeek、本地模型
- 每个 Agent 可以单独配置不同的 Provider
- 配置保存到 localStorage 或数据库

**UI 设计**：
```
┌─────────────────────────────────────────┐
│  设置                                   │
├─────────────────────────────────────────┤
│  默认 Provider: [DeepSeek ▼]           │
│                                         │
│  API Key: [*********************]       │
│  Base URL: [https://api.deepseek.com]   │
│                                         │
│  Agent 独立配置（可选）:                 │
│  ┌─────────────────────────────────┐   │
│  │ Router:  [跟随默认 ▼]           │   │
│  │ Tutor:   [跟随默认 ▼]           │   │
│  │ Examiner:[跟随默认 ▼]           │   │
│  └─────────────────────────────────┘   │
│                                         │
│  [保存配置] [测试连接]                  │
└─────────────────────────────────────────┘
```

**技术方案**：
- 前端：React Context + localStorage
- 后端：API 路由 `/api/settings`
- 类型安全：Zod schema 验证
